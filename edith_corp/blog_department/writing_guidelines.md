# ライティングガイドライン改訂版
*2026年2月4日 ユーザーフィードバックに基づく更新*

## 重要方針

### 1. ビジネス目的の明確化
**最終目的：価値あるAI活用情報の提供**
- ✅ 読者に本当に役立つAI活用法を紹介
- ✅ 実践的で今すぐ使えるテクニック
- ✅ 成功事例・失敗事例から学ぶ
- ❌ 露骨な宣伝・誘導はしない
- 💡 良質なコンテンツが自然に信頼を生む

### 2. 成田悠輔風の正しい理解

#### 口調・語尾
**❌ 間違った例：**
- 「〜だ」「〜である」（断定的・説教臭い）
- 「現実を見ろ」（上から目線）
- 「〜すべきだ」（命令調）

**✅ 正しい例：**
- 「〜ですよね」「〜じゃないですか」（親近感）
- 「〜かもしれません」（柔らかい提案）
- 「〜という感じです」（さらっと）
- 「〜みたいな」（カジュアル）

#### トーンの本質
**笑顔でさらっと毒を吐く**
- にこやかに痛いところを突く
- 深刻ぶらずに本質を語る
- ユーモアで包んだ鋭い指摘
- 読者を責めずに共感しつつチクリ

## 記事構成テンプレート（改訂版）

### 導入部（見出しなし）
**必須要素：**
- 「こんにちは、Room8の鶴田です！」から開始
- 挨拶の後、記事テーマに自然につなげる

**パターン例（テーマに応じて柔軟に変更）：**
```
こんにちは、Room8の鶴田です！
最近、〜〜な相談が多くて...

こんにちは、Room8の鶴田です！
この前面白いことがあって...

こんにちは、Room8の鶴田です！
〜〜について質問されることが増えてるんですけど...
```

### 問題提起
```
「確かにFAXとか使ってる会社も多いですよね。」
「でも実は、AIって意外と基礎ができてなくても使えちゃうんです。」
「むしろ両方同時にやっちゃった方が楽かもしれません。」
```

### 解決策提示
```
「例えばChatGPTなら、明日から使えますし。」
「FAX廃止は3ヶ月かかるけど、AI導入は1日でできちゃいます。」
「Room8では実際にこんな感じでやってます...」
```

### 締めくくり（自然な終わり方）
**基本パターン：**
```
「こういう使い方もあるので、試してみると面白いかもしれません。」
「実際にやってみたら、意外と簡単でしたよ。」
「他にも色々な活用法があるので、また紹介しますね。」
```

**シリーズ予告（自然に）：**
```
「次回は『Excel効率化の裏技』について書こうと思います。」
「今度は『ChatGPT活用事例』の話をしますね。」
「『AI導入の具体的手順』についても面白い話があるので、また書きます。」
```

**❌NG表現：**
- 「辛辣に解説します」→ 自分で辛辣とは言わない
- 「徹底解説」「完全攻略」→ 大げさな表現は避ける
- 「お楽しみに」→ かしこまりすぎ

## 具体例比較

### Before（問題のある書き方）
> 基礎もできてないのにAI導入って本末転倒だ。現実逃避してないで足元見ろ。

### After（正しい書き方）
> 基礎もできてないのにAI導入って、普通に考えたら順番逆ですよね。でも実は、AIの方が導入簡単だったりするんです。FAX廃止に3ヶ月かけるより、ChatGPT使い始める方が早いかもしれません。

## コンテンツマーケティングの本質

1. **読者の課題に共感**
   - 「FAXまだ使ってる会社、多いですよね」

2. **実用的な解決策を提供**
   - 「ChatGPTならこんな使い方ができます」
   - 「具体的な手順はこうです」

3. **実践例を紹介**
   - 「実際にやってみたら、月5万円削減できました」
   - 「失敗したケースもあって、それは〜」

4. **価値提供で終わる**
   - 「参考になれば幸いです」
   - 露骨な誘導なし、良質なコンテンツが信頼を生む

## NGワード・表現（文章全体）

### 自己評価系（絶対NG）
- 「辛辣に」「毒舌で」「厳しく」→ 自分では言わない
- 「徹底的に」「完全に」「詳しく」→ 大げさな表現避ける
- 「お楽しみに」「ご期待ください」→ かしこまりすぎ

### 語尾・トーン系
- 「べきだ」「はずだ」→「かもしれません」「みたいです」
- 「現実を見ろ」→「現実的にはこんな感じですよね」
- 「本末転倒だ」→「順番逆っぽく見えますけど」
- 「無駄だ」→「もったいないかも」

### 成田悠輔の本質
- 辛辣なことを言うけど、自分で「辛辣」とは言わない
- 毒があるけど、自分で「毒舌」とは表現しない
- 鋭い指摘をするけど、さらっと自然に
